Версия 2.46.1 (05.05.2022)

  • Исправлены ошибки:
    • «Поле объекта не обнаружено (Сумма)» — возникала при подготовке заказа к загрузке;
    • «Не получен ответ от плагина за разумное время» — появлялась при формировании подтверждения заказа;
    • «Поле объекта не обнаружено (НДС)» — возникала при формировании подтверждения заказа с использованием генератора ФЭД;
    • «Поле объекта не обнаружено (кол-во)» — появлялась при загрузке документа в 1С, если в документе не указано количество позиций номенклатуры.

Скачать:

  • для обычных форм (EPF, 1,2 МБ);
  • для управляемых форм (EPF, 1,25 МБ).

Версия 2.46 (26.04.2022)

  • В разделе «Продажа» размещен подраздел для электронных транспортных накладных. Реализовано для конфигураций 1С, в которых есть форма документа «Транспортные накладные»:
    • «Комплексная автоматизация», редакции 2;
    • «Управление торговлей», редакции 11;
    • «ERP Управление предприятием».
  • Новый механизм заполнения реализации из актов приемки — при повторной загрузке актов новые данные появятся только в тех графах, которые заполнены в акте приемки. Информация из других полей удаляться не будет.
  • Обновление обработки загрузится автоматически, если включено автообновление. Когда установка закончится, появится окно с предложением перезапустить систему.
  • Информация о наличии новых версий отобразится в уведомлении.
  • Появилась возможность скачать любую актуальную версию обработки из истории изменений.

Скачать:

  • для обычных форм (EPF, 1,2 МБ);
  • для управляемых форм (EPF, 1,25 МБ).
