Внешняя обработка СБИС обновлена на версию 2.44
Новое в версии
  • Реализовано создание дополнительных свойств EDI для обмена электронными документами с торговыми сетями. Новый функционал доступен из раздела "Настройки" - "Сервис" - "Настройка EDI". Для создания дополнительных свойств необходимо отметить сети, с которыми вы работаете, и нажать "Создать свойства". Настройки реализованы для десяти самых популярных торговых сетей.
